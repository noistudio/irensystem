  Мануал по установке и настройке фронта для проектной системы.
  1. переименовываем файл  vue.custom.config.example.js в vue.custom.config.js
  и прописываем свои данные

  2.  переименовываем файл vue.register_service.config.example.js в vue.register_service.config.js
        и прописываем свои данные

  3.Меняем текст или там стили и т.д
  4. Собираем
  npm  install
  npm run build
